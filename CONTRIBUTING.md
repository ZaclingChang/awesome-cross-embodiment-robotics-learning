# Contributing

Thanks for contributing.

This repository focuses on papers that are genuinely useful for studying:

- cross-embodiment robotics learning
- policy transfer across embodiments
- human-to-robot and human-to-humanoid transfer
- humanoid-to-humanoid transfer
- humanoid-to-biped or broader morphology generalization
- unified action / interface design
- evaluation protocols for unseen embodiments

## What to include

Please include:

- paper title
- link
- year
- 1-line takeaway
- tags from the README legend

## What not to include

Please avoid adding papers that are:

- unrelated single-task papers with no relevance to shared-policy learning
- generic VLM / LLM papers without embodied control implications
- hardware-only papers with no learning or control insight

## Suggested PR format

```md
- [Paper Title](link) `[H][M][OP][R]`
  One-line takeaway.
```
